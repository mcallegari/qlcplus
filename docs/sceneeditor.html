Scene Editor

The scene editor, as its name suggests, is used to edit scene functions. The editor is divided into tabs; with the first, General tab, you control the list of fixtures that take part in the scene, the scene's name and the scene's fade speeds. All subsequent tabs are used to control the individual channel values for each fixture.

General Tab Controls

Note: The toolbar on top of the window is disabled completely while the General tab is visible.

Scene name Change the name of the scene.
Add an existing Fixture to the scene.
Remove the selected Fixture(s) from the scene.
Enable all / Disable all Enable all channels of all selected fixtures.
Speed Change the scene's speed settings.

Fixture Tabs

Each fixture is represented by its own tab that contains sliders for each of the fixture's channels. Each channel can be enabled or disabled with a check box on top of the channel unit. The value of each channel can be set either by typing the value to the edit box on top of the slider or by moving the slider. Channels that provide multiple functions such as gobos, colors, etc. have also a button above the channel slider; this button can be used to directly select a specific function or capability provided by that channel.

Channel enabled/disabled status

If a channel has not been enabled, the scene will not touch that particular channel's value ever.

If a channel has been enabled, the scene will change that channel's value to a value defined in the scene. This is useful, for example, when you wish to control only the dimmer channel of a scanner fixture; you wouldn't want the scene to touch the scanner's pan, tilt, colour or gobo channels when you just want to fade in or fade out with the dimmer channel.

Controls

Enable all channels from the current fixture.
Disable all channels from the current fixture.
Copy the values (and enabled/disabled states) of all channels in the current fixture to the clipboard.
Paste values (and enabled/disabled states) from clipboard to the current fixture.
Copy all values (and enabled/disabled states) of all channels to all other fixtures taking part in the scene.
Launch a colour tool to select a specific colour and set that colour to the current fixture; This feature is enabled only for fixtures that are capable of CMY/RGB colour mixing.
