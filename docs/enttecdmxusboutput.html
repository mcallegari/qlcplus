1 Introduction

The DMX USB plugin supports a variety of FTDI-based USB-to-DMX devices:

  • DMXKing USB DMX512-A
  • DMXKing ultraDMX micro
  • DMXKing ultraDMX Pro
  • Enttec DMXUSB Open
  • Enttec DMXUSB Pro
  • Enttec DMX USB PRO Mk2
  • DMX4ALL USB-DMX STAGE-PROFI MK2
  • FTDI USBCOM 485+

2 Requirements

2.1 Linux

On all Linux distributions, you need to install libftdi. If you install QLC+ with the Ubuntu Software Center or some other automatic installer tool, this library will be installed automatically for you.

2.2 Mac OS X

On the Apple Mac OS X, you don't need any drivers at all since QLC+ uses the OS X native USB interface. Installing the D2XX drivers should cause no harm, but DO NOT INSTALL VCP drivers as they will definitely interfere with QLC+. If you have previously installed the VCP drivers, consult the FTDI installation guides on how to uninstall them.

2.3 Windows

On Microsoft Windows, the plugin needs the latest D2XX drivers from FTDI. Consult the FTDI installation guides on how to install the drivers. DO NOT INSTALL VCP drivers as they will probably interfere with the D2XX interface.

3 Tuning

It is possible to change the DMX frame frequency for Enttec Open (and like) devices with a hidden settings key on each platform. The key tells QLC+ how many times each DMX frame (512 channels) should be sent to the universe per second. A value of "30" means 30 times per second (30Hz).

3.1 Linux

  • Open gedit, kedit, nano, vim or some other PLAIN text editor
  • Open the following file from your home directory: ".config/qlcplus/Q Light Controller Plus.conf"
  • Place the following text to the very end of the file (and replace 30 with the value you want):
    [enttecdmxusbopen]
    frequency=30

3.2 Mac OS X

  • Install TextWrangler from Mac App Store, it's completely free.
  • Open the following file with TextWrangler from your home directory: "Library/Preferences/net.sf.Q Light Controller Plus.plist".
  • Scroll to the very bottom of the file, just above the last "</dict>" row and put the following text there (replace 30 with the frequency you want):
    <key>enttecdmxusbopen/frequency</key>
    <string>30</string>

3.3 Windows

  • Open the Windows Registry Editor
  • Navigate to HKEY_CURRENT_USER/Software/qlc.sf.net/Q Light Controller Plus
  • Create a new group and name it "enttecdmxusbopen"
  • Create a new decimal integer key "frequency" and set its value to the desired DMX frame frequency
