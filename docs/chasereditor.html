Chaser Editor

The chaser editor, as its name suggests, is used to edit chaser functions.

Controls

Chaser name Change the name of the chaser.
Add an existing Function to the chaser, using the Select Function dialog. The function are inserted after the currently selected step.
Remove the selected step from the chaser.
Raise the selected step above/before the previous step.
Lower the selected step below/after the next step.
Run Order
  • Loop: Run thru the steps over and over again.
  • Single Shot: Run thru the steps once and then stop.
  • Ping Pong: Run thru the steps over and over again, reversing direction at both ends.
Direction
  • Forward: Run thru the steps from start to end; 1, 2, 3...
  • Backward: Run thru the steps from end to start; ...3, 2, 1
Speed Change the chaser's speed settings.
